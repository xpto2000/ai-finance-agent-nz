# 💸 AI Finance Agent NZ

AI-powered tool to analyse New Zealand bank transactions.

Upload CSV files and get:
- Automatic transaction classification (AI)
- Spending breakdown charts
- Simple financial insights

## Tech
- Python
- Streamlit
- Pandas
- Plotly
- CrewAI

## Run locally

pip install -r requirements.txt  
streamlit run app.py  

## Notes
- Built for NZ bank data formats
- Uses AI for classification (not hardcoded rules)
